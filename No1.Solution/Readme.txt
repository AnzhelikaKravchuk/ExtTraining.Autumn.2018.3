Идея состоит в создании листа из делегатов.
При добавлении нового метода ограничений, он заносится в лист.
После этого, когда понадобится выполнить проверку пароля, лист будет пройден 
и каждый делегат выполнится для входного пароля.
Если хотя бы один из делегатов вернул false то весь метод возвращает false.

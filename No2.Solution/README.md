Для мделирования работы отправки сообщений был создан тип StockInfoManager, отвечающий за отправку сообщений клиента, подписавшимся на событие. Всякий раз, когда вызывается метод Market класса Stock, вызывается метод SimulateNewStockInfo класса StockInfoManager, инфрмирующий всех подписавшихся клиентов об изменениях на рынке.
